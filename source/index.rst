===========================================
Welcome to SanctionSightPy documentation!
===========================================

**SanctionSightPy** is a Python library designed to provide structured access to global sanctions data from multiple regulatory bodies, including OFAC (U.S.), EU, Canada, UK-OFSI, DFAT (Australia), and SECO (Switzerland). The package will enable seamless querying, filtering, and integration of sanctions lists into compliance workflows.

----------------------------------

**Prerequisites**

To get started with the SanctionSightPy library, ensure the following requirements are met:

* Python 3.10+

--------------

**Target Audience**

This library is intended for developers familiar with Python and sanction data.

To get started, refer to the :doc:`get_started` section for installation instructions and requirements.
